# 🍽 MenuCraft Ethiopia
### Digital Menu Builder & QR Code Generator

> **Free. Serverless. Private. Built for Ethiopian restaurants.**

A beautifully designed, client-side web application that lets restaurant and café owners create a professional digital menu and generate a scannable QR code — with zero backend, zero database, and zero cost.

---

## ✨ Features

| Feature | Details |
|---|---|
| 🔒 **100% Private** | All menu data is encoded inside the QR code itself. Nothing is ever uploaded to a server. |
| 🆓 **Zero Cost** | Runs entirely on GitHub Pages — no hosting fees, no backend, no database. |
| 📱 **Instant Customer View** | Customers scan the QR → their phone opens a beautiful, styled menu page. |
| 🇪🇹 **Amharic Support** | Full UTF-8 / Ethiopic Unicode compatibility. No scrambled characters. |
| 🖨 **Print-Ready QR** | Download a crisp `.png` image to print on table cards or posters. |
| ⚡ **Live Preview** | See your menu update in a phone mockup as you type — in real time. |

## 🙋 FAQ

**Q: Can customers pay through the QR code?**
A: The CBE account number is displayed on the menu page so customers know where to transfer. You can also add your Telebirr number in the slogan field.

**Q: What happens if I update the menu?**
A: Simply re-enter your menu, generate a new QR code, and replace the old printed one. Each QR encodes the full menu.

**Q: How many items can I add?**
A: Practically unlimited, but QR codes become harder to scan with very large data. We recommend under 30 items for reliable scanning.

**Q: Is this free forever?**
A: Yes — GitHub Pages is free for public repositories indefinitely.
